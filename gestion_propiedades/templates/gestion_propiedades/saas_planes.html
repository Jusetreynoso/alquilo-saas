¿Cómo se calcula la Facturación SaaS?

Todos los planes en la plataforma operan bajo un modelo híbrido (Base + Pay-As-You-Grow). Al momento del corte, el robot de cobros aplicará automáticamente esta fórmula:

Factura = Tarifa Base del Plan + ($1.00 × Propiedad Activa) + ($1.00 × Asistente Extra)

Ejemplo de Modelos:
• Si quieres un modelo 100% Por Uso: Crea un plan con Tarifa Base $0.00.
• Si quieres cobrar una membresía de plataforma, asigale la tarifa base deseada y el sistema seguirá sumando los volúmenes de uso.
